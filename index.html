About

你善於觀察生活中的問題嗎？你想要把腦中各種有趣的想法實現出來嗎？

MakeNTU 提供給熱衷於動手設計與實作的人一個恣意揮灑創意的舞台，讓Maker們在Open Source以及各種硬體支援的環境裡盡情發揮，動手解決生活周遭所遇到的各種問題！

3/24, 3/25的教學工作坊，帶領大家實作AI之軟體API與運用相關雲端服務。3/31, 4/1完整24小時的黑客松競賽則讓大家自由發揮創意並實作成品，並在過程中相互交流激發創客潛能！只要動手，人人都可以成為Maker！

大會題目：人工智慧 AInsight

近年來，人工智慧的蓬勃發展，無論是語音辨識、電腦視覺、對局理論，甚至遠至交通與農業問題，都可看見人工智慧的應用。科技部更將2017年訂為人工智慧元年，積極爭取將人工智慧的發展列入前瞻建設之中。有鑑於該領域的發展潛力與應用廣泛，本次大會主題也搭上這股潮流，希望能做為創客松創新發想與實做的起點。

作為一位科技公民與創客，我們期許參賽者從自身出發，利用各種雲端服務並結合硬體設備，實現一個能發揮人工智慧技術的構想，並實作出一個足以傳達理念的「實體原型」，進而運用至日常生活中，來提升與改善人類的生活環境。

Platforms

Arduino

目前市面上最常見的的開發電路板，雖然開發過程簡單，但軟硬體都是開放的，參考資料多，因此上手非常快，能做出許多令人驚艷的互動式裝置。

Raspberry Pi 2 & 3

麻雀雖小，五臟俱全。一塊小小的 Raspberry Pi ，卻能堪稱一台小型電腦，擁有開放的架構、親民的價格、豐富的應用，能讓你實現各種有趣的想法。

STM32 Nucleo & extension board

STMicroelectronics所推出的開發版，具備與Arduino相容的腳位，也能夠輕易進行多種特殊應用附加元件的擴充。此外也能搭配多種開發環境，讓使用者能夠快速打造開發環境，嘗試新的構想。

Intel Edison & Genuino & Galileo

Edison 與 Arduino 的程式碼和外接模組相容，是運行 Linux 的小型電腦。相較於偏軟體的 RPi，它們主要用於開發硬體裝置，更內建了 Bluetooth、Wi-Fi，適用於穿戴裝置與物聯網。Genuino 101 則是進階版的 Arduino，使用相同的 IDE，同時內建六軸陀螺儀和 Bluetooth LE。

ASUS Tinker Board

一起動手做！這是我們深植於心中的理念，Tinker Board是一台小型電腦，透過開放的軟硬體架構，豐富的連接介面，為使用者設想的友善外觀設計，讓新奇作品可以輕鬆、簡單的被打造而成；而內建的千兆網路、Wi-Fi與Bluetooth，也開啟了物聯網應用的充沛可能性。更好的是，其搭載影音平台等級的四核心處理器、HD高音質輸出晶片，更造就Tinker Board適用於家用影音播放中心。Tinker Board不僅讓我們再次愛上DIY，也讓自造者們在新一代的平台上實現夢想!

LinkIt 7697 & 7688

市售最小的系統單晶片，適合開發物聯網、穿戴式裝置，同時又擁有 MediaTek Cloud Sandbox，提供遠端控制、資料儲存、API等豐富工具。內配 Wi-Fi、Bluetooth、衛星定位等等，2016 HITCON CTF 中的隊伍狀態燈正是使用 LinkIt 設計出來的！

Google Cloud API

Google雲端平台提供的機器學習環境，Google提供了多種程式語言的客戶端函式庫，透過這些函式庫，能夠使用如圖像辨識、自然語言處理等多種的AI的引擎，快速建立起人工智慧的環境。

Microsoft Azure API

Microsoft的Azure API，可提供虛擬機與巨量資料的機器學習服務，也支援多種GPU和深度學習框架，能夠讓用戶快速的建構AI的應用程式。

Amazon Web Services (AWS)

相信大家都聽過 AWS 的服務，像是 EC2 server，以及不需 server 就能跑的 AWS Lambda。AWS 的服務包山包海，storage、database、machine learning...... 應有盡有！

FAQ

MakeNTU？

2017 MakeNTU 創客松將舉辦於 3/31~4/1 兩天一夜。參賽者於比賽會場連續 24 小時腦力激盪並實作成品，依據MakeNTU大會所提供的主題方向，參賽者可以盡情發揮創意並動手實踐，你可以現在就和你的隊友開始構思題目！若毫無頭緒的話，可以看看 MakeNTU 提供的主題相關資訊，會給你一些靈感的！

另外，我們今年還會在 3/24~3/25 舉辦「新創論壇」&「工作坊」，讓參賽者可以更加熟悉題目和開發工具喔！

Expo/Demo？

在 MakeNTU 第二天的Expo時段，將開放參賽者與一般民眾自由參觀各組作品，各組除了展示作品功能，也可以輔以簡單的簡報說明，讓大家愛上你們的idea！在Expo時段除了參賽者互評選出人氣獎之外，同時評審也會評選出Top 8參與最終的Demo。被選出的優勝組別將於 Final 時段上台向所有人 Demo 成品，並在活動尾聲由所有評審共同選出榮獲四大獎的得獎隊伍。

誰具備報名資格？

凡是全台大專院校的學生都具備有參賽資格，必須以四人為一組報名！不管從前的你有沒有參與過黑客松，MakeNTU絕對是你不容錯過的全新創客體驗！

活動的報名費用？

完！全！免！費！ 除此之外我們還會提供三餐、宵夜、以及無限制的飲料與零食，讓Maker們能盡情創作而無後顧之憂！不過為了避免報名隊伍於活動當天缺席，我們將需要錄取隊伍每隊繳交 1000 元之保證金，若隊伍全程參與完畢（不含創新論壇、工作坊），大會將於活動結束後退還保證金。

我需要會什麼呢？

完全不用！！當然如果有簡單C/C++的程式基礎會比較快上手，不過並不需要過於擔心自己的能力！MakeNTU除了會提供一系列的實戰工作坊之外，活動現場也會有世界級的Mentor於現場協助大家。MakeNTU只是一塊敲門磚，讓大家發掘心中的創客潛能，只要動手，人人都可以成為Maker！

我需要帶什麼呢？

除了參賽者需自行攜帶的筆電或桌電之外，我們會準備各種開發板、電子元件與耗材供參加者使用，並提供每隊最多 1000 元的額度上限，供各組自行採購所需要的零件與材料（三創隔壁即光華商場，所購買的物品需打上大會統編並於活動結束後歸還）。為確保活動公平性，參賽者所使用的材料皆須由大會提供，或是由一千元額度所購買，禁止自行攜帶材料前往。若參賽者有其他材料需求，可以事先通知主辦單位，我們會盡力備齊大家所需的所有材料喔！
